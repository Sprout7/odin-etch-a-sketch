## Etch-a-Sketch (The Odin Project)

The goal is to build Etch-a-Sketch using HTML, CSS, and JavaScript

## Project Pseudocode

- Create container div in HTML to hold grid
- Set container size to 960px by 960px
- Use JavaScript to generate 16 x 16 grid (256 total square divs)
- Append each square div to container
- Use flexbox wit wrapping to display squares as grid
- Size each square evenly so grid fits inside container
- Add hover effect so squares change color when mouse passes over them
- Add button that allows the user to reset grid with new size
